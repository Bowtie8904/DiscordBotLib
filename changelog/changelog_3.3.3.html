Discord Bot Lib v3.3.3

Bot class

  • Improved Bot#sendMessage(), Bot#sendListMessage() and Bot#sendPlainMessage() methods
    • Suppressed spammy errorlogging
    • Removed all permission checks. If a MissingPermissionException occours the moothods will simply return null
  • Added controller overrides to create a bot with the recommended amount of shards
  • Added errorlog suppression to Bot#changePlayingText()

Command class

  • Switched overrides and cooldowns to a map system to allow the usage of a single command instance instead of one for each guild

CommandEvent class

  • Added methods for the usage of parameters inside the command string
  • Parameters are indicated by a dash (-) followed by a unique name an equal sign (=) and the value

ChannelLogger class

  • Added support for filter
  • Filter will cause lines which contain at least one of the set strings to not be sent to the channel

OfflineHandler class

  • Added an option to turn off file logging
